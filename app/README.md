# 🎵 Spotify Music Trends Analyser

A data analytics dashboard built with Python and Streamlit to analyse trends, genres and mood of 114,000 Spotify songs.

## 📊 Features
- Top 10 most popular songs
- Genre vs music features (energy, danceability, acousticness)
- Mood (valence) comparison across genres

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Streamlit

## 📁 Dataset
Spotify Tracks Dataset from Kaggle (114,000 songs, 21 features)

## 🚀 How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the dashboard: `python -m streamlit run app/dashboard.py`
